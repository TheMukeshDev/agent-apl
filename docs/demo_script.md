# SportSphere AI - 2-Minute Demo Pitch & Script

This demo script outlines the hackathon presentation and live product walk-through for **SportSphere AI**.

---

## 🎙️ The Pitch (30 Seconds)

**Presenter:**
> "Good afternoon, judges. Lucknow is experiencing a massive sports boom, yet players are struggling to find and book arenas. Fragmented WhatsApp groups, hidden pricing, and manual phone bookings lead to empty courts and frustrated players.
>
> Introducing **SportSphere AI**—a unified agentic sports concierge for Lucknow. By matching natural language commands to a specialized agent workflow, we let users discover, compare, and instantly book sports infrastructure in seconds."

---

## 🖥️ Live Walkthrough Script (90 Seconds)

### Step 1: Intelligent Search & Discovery (30s)
1. **Action**: Select "Badminton" and "Gomti Nagar" in the UI search filters, then click **Find Venues**.
2. **Narration**: 
   > "Let's say we are looking for a badminton court near Gomti Nagar. With our Discovery Agent, we filter venues in real-time. But what if we want to talk to our system naturally? We can use our AI Agent Dialog box.
   > Let's ask: *'Find beginner badminton under ₹200 near Gomti Nagar.'*"
3. **Action**: Submit the query to the Chat box.
4. **Narration**:
   > "On the backend, our LLM Parser extracts the parameters: Sport is Badminton, Budget limit is 200, Location is Gomti Nagar, Skill level is Beginner. The Recommendation Agent scores each venue out of 100 based on affordability, location, and skill compatibility, rendering **Gomti Nagar Badminton Court** as the top match."

### Step 2: Neighborhood Coordinate Grid Map (30s)
1. **Action**: Point to the **Lucknow Sports Map Grid** section.
2. **Narration**:
   > "To solve navigation confusion, we've built a custom coordinate map grid of Lucknow. You can see live coordinate pins for Gomti Nagar, Chinhat, and Aliganj representing active arenas. Hovering over a pin reveals the arena details, and clicking it displays the venue details drawer instantly."

### Step 3: Instant Booking & Cancellation Loop (30s)
1. **Action**: Click **Book Now** on the Gomti Nagar court, enter name "Mukesh Kumar", select date, and click **Confirm Booking**.
2. **Narration**:
   > "Once we decide, booking is a single click. Our Booking Agent automatically generates a unique reference ID—like `SS-F38B92`—and stores the confirmation in memory. If our plans change, we can click **Cancel Booking** under the Active Bookings section, which frees up the slot on the server immediately."

---

## 🏁 Conclusion (15 Seconds)

**Presenter:**
> "By combining conversational AI discovery, automated recommendation scoring, custom map grids, and real-time slots booking, SportSphere AI elevates sports connectivity in Lucknow. Thank you!"
